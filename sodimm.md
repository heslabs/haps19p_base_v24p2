# DDR4 SODIMM



#### DDR Connectors
* MIG1 : on board DDR
* MIG2 : JX1~3 DDR card, Voltage 1.2V
* MIG3 : JX6~8 DDR card, Voltage 1.2V
* MIG4 : JX13~15 DDR card 
* MIG5 : JX22~24 DDR card, Voltage 1.2V
